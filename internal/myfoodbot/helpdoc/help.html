Бот MyFoodBot предназначен для учета веса и потребляемых калорий

Взаимодействие с ботом осуществляется посредством команд

Бот работает в таймзоне Europe/Moscow, настроек таймзоны под пользователя на данный момент нет

Формат команд

Команды задаются в формате значений, разделенных запятой: command,subcommand,arg1,arg2,...

Если после или до запятых есть пробелы, то они удаляются

Команды и подкоманды задаются в нижнем регистре

Для получения руководства необходимо отправить: h

Для получения расчета калорий необходимо отправить: cc,<Пол 'm'|'f'>,<Вес (кг.)>,<Рост (см.)>,<Возраст (лет)>

Значения веса, роста, возраста > 0

Рассчитываемые активности:

Сидячая: если нет физических нагрузок и сидячая работа

Легкая: если есть небольшие пробежки или легкая гимнастика 1–3 раза в неделю

Средняя: если есть занятия спортом со средними нагрузками 3–5 раз в неделю

Полноценная: если есть тренировки 6–7 раз в неделю

Супер: если работа связана с физическим трудом, или есть тренировки 2 раза в день с силовыми упражнениями

Команды для управления пользовательскими настройками начинаются с: us

Установка настроек

Команда: us,set,<Лимит калорий>

Значение Лимита калорий > 0

Получение настроек

Команда: us,get

Команды для управления весом начинаются с: w

Установка веса

Команда: w,set,<Дата MM.DD.YYYY>,<Значение>

Значение веса > 0

Если дата пустая, то подразумевается текущая дата

Удаление веса

Команда: w,del,<Дата MM.DD.YYYY>

Если дата пустая, то подразумевается текущая дата

Вес за диапазон дат

Команда: w,list,<Дата С MM.DD.YYYY>,<Дата По MM.DD.YYYY>

Если дата пустая, то подразумевается текущая дата

Команды для управления едой начинаются с: f

Установка еды

Команда: f,set,<Ключ>,<Наименование>,<Бренд>,<ККал100>,<Бел100>,<Жир100>,<Угл100>,<Комментарий>

Ключ - уникальная строка для данной записи

Наименование - название еды

Бренд - производитель еды (необязательное поле)

ККал100 - значение ккал в 100г.

Бел100 - значение белков в 100г.

Жир100 - значение жиров в 100г.

Угл100 - значение углеводов в 100г.

Комментарий (необязательное поле)

Установка комментария для еды

Команда: f,sc,<Ключ>,<Комментарий>

Шаблон для установки параметров еды

Команда: f,st,<Ключ>

Поиск еды

Команда: f,find,<Шаблон>

Осуществляется поиск записей еды по совпадению шаблона в полях Ключ, Наименование, Бренд и Комментарий

Выводится не более 10 записей

Весь список еды

Команда: f,list

Расчет энергетической ценности еды

Команда: f,calc,<Ключ>,<Вес еды в гр.>

Удаление еды

Команда: f,del,<Ключ>

Нельзя удалить еду, которая уже используется в журнале приема пищи или бандле

Команды для управления бандлами с: b

Бандл - это группировка еды и других бандлов для быстрого добавления записей в журнал еды

Установка бандла

Команда: b,set,<Ключ бандла>,<Ключ еды:вес>,<Ключ дочернего бандла>,...

В бандл можно добавлять как еду так и другие бандлы (кроме самого себя)

В случае если указывается ключ еды, то вес > 0

Шаблон для установки параметров бандла

Команда: b,st,<Ключ бандла>

Список бандлов

Команда: b,list

Удаление бандла

Команда: b,del,<Ключ бандла>

Нельзя удалить бандл, который является дочерним для другого бандла

Команды для управления журналом приема пищи с: j

Установка записи приема пищи

Команда: j,set,<Дата MM.DD.YYYY>,<Прием пищи>,<Ключ еды>,<Вес гр.>

Варианты приема пищи: Завтрак, До обеда, Обед, Полдник, До ужина, Ужин, Перекус

Если указан какой-то другой прием пищи (или пустой), то сохраняется прием пищи - Перекус

Ключ еды - значение ключа из списка еды

Если дата пустая, то подразумевается текущая дата

Установка бандла для записи приема пищи

Команда: j,sb,<Дата MM.DD.YYYY>,<Прием пищи>,<Ключ бандла>

Ключ бандла - значение ключа из списка бандлов

Варианты приема пищи: Завтрак, До обеда, Обед, Полдник, До ужина, Ужин, Перекус

Если указан какой-то другой прием пищи (или пустой), то сохраняется прием пищи - Перекус

Если дата пустая, то подразумевается текущая дата

Удаление записи приема пищи

Команда: j,del,<Дата MM.DD.YYYY>,<Прием пищи>,<Ключ еды>

Варианты приема пищи: Завтрак, До обеда, Обед, Полдник, До ужина, Ужин, Перекус

Если указан какой-то другой прием пищи (или пустой), то будет подразумеваться - Перекус

Ключ еды - значение ключа из списка еды

Если дата пустая, то подразумевается текущая дата

Удаление приема пищи

Команда: j,dm,<Дата MM.DD.YYYY>,<Прием пищи>

Варианты приема пищи: Завтрак, До обеда, Обед, Полдник, До ужина, Ужин, Перекус

Если указан какой-то другой прием пищи (или пустой), то будет подразумеваться - Перекус

Если дата пустая, то подразумевается текущая дата

Копирование приема пищи

Команда: j,cp,<Дата откуда MM.DD.YYYY>,<Прием пищи>,<Дата куда MM.DD.YYYY>,<Прием пищи>

Для копирования указывается дата и прием пищи откуда копировать и дата и прием пищи куда копировать

Если в дате назначения уже есть записи у указанного приема пищи, то будет ошибка копирования

Приемы пищи источника и назначения могут быть разными

Если дата пустая, то подразумевается текущая дата

Отчет по приему пищи за дату

Команда: j,rm,<Дата MM.DD.YYYY>,<Прием пищи>

Если дата пустая, то подразумевается текущая дата

Ежедневный отчет

Команда: j,rd,<Дата MM.DD.YYYY>

Если дата пустая, то подразумевается текущая дата

Еженедельный отчет

Команда: j,rw,<Дата MM.DD.YYYY>

По дате определяется начало недели, относительного которого формируется отчет

Если дата пустая, то подразумевается текущая дата

Отчет по ККал за период

Команда: j,rr,<Дата С MM.DD.YYYY>,<Дата ПО MM.DD.YYYY>

Если дата пустая, то подразумевается текущая дата

Шаблоны команд по приему пищи за дату

Команда: j,tm,<Дата MM.DD.YYYY>,<Прием пищи>

Если дата пустая, то подразумевается текущая дата

Средний вес приема пищи за год

Команда: j,fa,<Ключ еды>

Ключ еды - значение ключа из списка еды

Год рассчитывается как <текущая дата - 1 год, текущая дата>
